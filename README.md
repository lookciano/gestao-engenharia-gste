# Portal de Engenharia GSTE

Este repositório centraliza as ferramentas de gestão utilizadas pelo time de engenharia da GSTE. O portal oferece acesso rápido e organizado às plataformas de controle de documentos e acompanhamento de atas.

## 🚀 Funcionalidades

- **Página Inicial Centralizada:** Interface moderna baseada na identidade visual corporativa (tons de verde petróleo e cinza).
- **Controle de Documentos de Engenharia:** Visualização e filtragem de documentos técnicos, com dashboards integrados.
- **Acompanhamento de Atas de Reunião:** Monitoramento de compromissos e restrições (Acesso protegido por senha).

## 🔒 Segurança

A plataforma de **Acompanhamento de ATA** possui acesso restrito. Solicite a senha ao administrador do sistema para acessar os dados.

## 🛠️ Como utilizar

Para hospedar este portal no GitHub Pages:

1. Suba os arquivos deste diretório para um repositório no seu perfil do GitHub.
2. Vá em **Settings** > **Pages**.
3. Em **Branch**, selecione `main` e a pasta `/ (root)`.
4. Clique em **Save**.
5. O link do portal será gerado automaticamente pelo GitHub.

## 📁 Estrutura de Arquivos

- `index.html`: Portal de entrada.
- `documentos.html`: Plataforma de Controle de Documentos.
- `ata.html`: Plataforma de Acompanhamento de Atas.
